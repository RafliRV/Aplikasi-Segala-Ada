Deskripsi Aplikasi:
  Aplikasi ini menyediakan 5 program dalam 1 Aplikasi.
  Untuk mengakses aplikasi ini,diperlukan username dan password.
  username  :admin
  password  :1234
  Berikut daftar program yang ada di aplikasi ini:
    1.Karyawan PT.CAHAYA JAYA PRATAMA
      Dalam program ini,program akan minta input data karyawan yaitu nama dan total penjualan.
      Kemudian akan menunjukkan komisi yang dia dapatkan.
    2.Studio Al Musik
      Program ini meminta input data tentang group yang akan menyewa alat musik dari studio Al Musik.
      Nanti akan ditunjukkan berapa total yang harus dibayar
    3.Kafe Kekinian
      Program ini akan meminta input kode paket makanan dan juga jumlah pembelian beserta kode kasir.
      Lalu nanti program akan menunjukkan bill nya.
    4.Warnet Kece
      Program ini akan meminta data nama pengunjung warnet,berapa lama dia rental warnetnya.
      Lalu kemudian akan menunjukkan berapa yang harus dibayarnya.
    5.Biodata Pribadi
      Program ini akan meminta input data diri,lalu akan menunjukkan perkenalan dengan data yang tadi diinput.
